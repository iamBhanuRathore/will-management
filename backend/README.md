# backend

This is the backend service for the Will Management application. It is built with Express.js, TypeScript, and Prisma.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/)
- [PostgreSQL](https://www.postgresql.org/)

### Installation & Setup

1.  **Install dependencies:**

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.16. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
